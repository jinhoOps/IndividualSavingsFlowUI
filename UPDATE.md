## 운영 원칙
- 개발 시작 전에 항상 `UPDATE.md`를 먼저 확인한다.
- 모든 변경 사항/개발 사항은 작업 종료 시 `UPDATE.md`에 요약 기록한다.
- 상세 로그가 길어지면 날짜 단위가 아니라 기능 단위로 압축해 유지한다.

## 압축 이력 (기존 내역)
### 2025-12-29 ~ 2026-01-26
- Vanilla HTML/CSS/JS 기반 개인 자산흐름 대시보드 MVP를 구축하고, 노드/링크 CRUD, 유효성 검증, 우선순위 정렬, 드로어 UI를 포함한 편집 워크플로를 완성.
- 흐름 시각화는 리스트/바형/Sankey 스타일로 확장했고, 필터/툴팁/레이아웃 보정/겹침 완화 작업을 반복 적용.
- 시뮬레이션 엔진은 월 단위 계산(수입 성장, 저축/투자 반영, 인플레이션/명목·실질 토글, 보유자산 가정)을 단계적으로 고도화.
- 입력/표현 체계를 만원 단위 중심으로 통일하고, 핵심 카드/요약 뷰/초기화 동작 등 UX를 지속 정리.

## 리디자인 이후 압축 (최초)
### 2026-02-25
- UI를 전면 리디자인: 복수 그래프/복잡 편집 중심에서 `핵심 카드 + 단일 Sankey + 계산 검증 표` 중심 구조로 재편.
- 입력 UX를 단순화: 간편 입력 + 고급 설정 접기 구조로 변경하고, 수입은 항목 추가/삭제 방식으로 전환.
- 생활비를 대표 하위항목(주거비/관리비/통신비/교통비/식비/기타) 자동 합산 방식으로 분리.
- 급여 기본값을 `290`으로 조정.
- 입력값 섹터를 화면 하단으로 이동하고, `입력값 섹터로 이동` 고정 버튼 추가.
- 부채 계산 로직 보정:
  - 월 이자 반영 후 상환 적용.
  - 실제 상환액 = `min(부채잔액, 설정 상환액, 가용현금)`.
  - 현금 부족분은 신규 차입으로 전환.
- 신뢰도 개선 지표 추가:
  - 카드/표에 `당월 이자`, `실제 상환액`, `신규 차입` 노출.
- 부채 누적 체감 이슈 보정:
  - 적자 구간에서 저축/투자를 고정 집행하지 않고, 가용현금 범위 내에서만 반영하도록 시뮬레이션 순서를 조정.
  - 생활비/부채상환 이후 현금이 부족할 때만 신규 차입이 발생하도록 모델링.
- Sankey 표현 개선:
  - 생활비 하위항목(주거비/관리비/통신비/교통비/식비/기타) 노출을 추가하고, 최종적으로는 `생활비 종합 노드 + 하위 상세(범례/툴팁)` 구조로 정리.
- Sankey/용어 보정:
  - `부족자금` 용어를 `신규차입`으로 변경해 의미를 명확화.
  - 생활비는 Sankey에서 종합 노드(`생활비`)로 표시하고, 하위항목은 범례/툴팁에 추가 노출.
- Sankey/용어 2차 정리:
  - `신규차입` 소스 노드를 제거하고, Sankey는 `월 배분총액 → 생활비(종합)/저축/투자/부채상환` 구조로 단순화.
  - 검증 표/카드 용어를 `부채 증가분`으로 통일.
- Sankey 표현 이슈 보정:
  - 생활비를 `종합 노드`로 유지하면서 차트 내부에서 `생활비 → 하위항목` 2단 분기로 추가 표시.
  - 하위항목은 범례/툴팁에서도 동일하게 상세 확인 가능하도록 동기화.
- 검증 가이드 보강:
  - 표 메타에 `첫 달 이자 > 실제상환`일 때 부채 증가 가능 안내 문구를 추가.
- 저축/투자 하위항목 확장:
  - 저축을 `청년적금`, `주택청약`으로 분리하고 월 저축은 자동합산(읽기 전용)으로 고정.
  - 투자를 `해외주식`, `ISA`, `금현물`로 분리하고 월 투자는 자동합산(읽기 전용)으로 고정.
  - Sankey에 `저축/투자 종합 노드 → 하위항목` 2단 분기를 추가해 생활비와 동일한 표현 체계로 통일.
  - 상세 항목 툴팁/범례 동기화 및 관련 런타임 참조 오류를 정리.
- Sankey 라벨 배치 조정:
  - 마지막 컬럼(최종 노드)만 라벨을 우측 표시로 유지.
  - 1단 포함 중간 컬럼 라벨은 좌측 표시로 변경.
- Sankey 톤 조정:
  - 생키 링크/노드 색상을 더 옅은 투명 톤으로 조정해 과한 채도를 완화.
  - 호버 시에도 기존보다 부드럽게 강조되도록 불투명도 범위를 조정.
- 기본값/입력 UX 업데이트:
  - 기본 입력값을 캡처 기준으로 조정:
    - 월 수입 `300`, 생활비(60/10/5/10/40/20), 저축(70/5), 투자(30/30/3)
    - 초기 현금 `100`, 초기 저축잔액 `3000`, 초기 투자잔액 `3000`, 초기 부채 `0`
    - 수입 증가율 `4.0%`, 생활비 증가율 `2.5%`, 저축 수익률 `3.0%`, 투자 수익률 `9.5%`, 부채 이자율 `4.2%`
  - 라벨 보강: `생활비 증가율(연, %, 기준금리)`로 부연 설명 추가.
  - 입력값 섹터 변경 시 즉시 반영 대신 하단 중앙 `적용/취소` 바를 표시하고, 사용자 확인 후 반영하도록 수정.
  - `적용/취소` 바는 실제 변경사항이 있을 때만 노출되도록 조정.
  - 적용 완료 시 `변경사항이 적용되었습니다.` 피드백이 잠시 표시된 뒤 자동으로 사라지도록 추가.
- 타이포그래피 조정:
  - 메인 제목(`h1`) 폰트를 `Gowun Dodum` 계열(`var(--font-body)`)로 변경.
