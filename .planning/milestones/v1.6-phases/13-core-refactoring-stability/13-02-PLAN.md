# list-renderer.js 모듈 추출 (3계층 모듈화)

<objective>
app.js 내의 순수 UI 렌더 함수 7종과 보조 힌트 함수 4종(약 130줄)을 apps/step1/modules/list-renderer.js로 추출하여 3계층(상태/헬퍼/UI) 모듈화를 완성합니다. step2의 renderers.js 패턴을 참조하여 state/dom을 직접 import하는 방식을 채택합니다.
</objective>

<must_haves>
## Goal-Backward Verification
- truth: 6종 핵심 헬퍼 함수(hasPendingChanges, markPendingChanges, commitImmediateInputs, getVisibleInputs, markDirty, markClean)가 리팩터링 후에도 app.js에서 정상 호출 가능하다
- truth: renderAll() 함수가 list-renderer.js에서 import한 renderCards, renderProjectionTable, renderInputHints를 호출하여 동일한 출력을 생성한다
- truth: renderItemList가 콜백으로 전달되는 경로(handleApplySmartAdd L237 등)에서 정상 동작한다
- truth: 추출된 모듈은 ES Module 기반이며 브라우저에서 빌드 없이 직접 실행 가능하다
- truth: XSS 방지를 위해 IsfUtils.escapeHtml 사용이 그대로 유지된다
</must_haves>
