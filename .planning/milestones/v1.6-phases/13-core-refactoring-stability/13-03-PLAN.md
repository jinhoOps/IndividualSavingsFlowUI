# 미사용 코드 정리 및 중복 함수 통합

<objective>
app.js 내의 미사용 임포트 9종, 미사용 함수 6종을 제거하고 중복 정의 3쌍을 통합하여 코드 위생을 확보합니다. REF-01(list-renderer.js 추출) 완료 후의 임포트 상태를 기준으로 정리합니다.
</objective>

<must_haves>
## Goal-Backward Verification
- truth: 제거 대상 임포트/함수를 제거한 후 app.js에서 런타임 ReferenceError가 발생하지 않는다
- truth: 중복 함수 통합 후 기존 동작이 동일하게 유지된다
- truth: 미정의 함수 4종(hasShareState, dismissViewModeGuide, switchToNormalMode, bindReadonlyAdvancedNavigation)은 삭제하지 않고 런타임 존재 여부만 확인한다
- truth: ui-controller.js에 이미 export된 함수로 중복 로컬 정의를 대체할 때, 호출 시그니처가 동일하다
</must_haves>
