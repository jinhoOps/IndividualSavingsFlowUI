# Requirements Archive — Milestone v1.0: 템플릿 기반 자산 흐름 시각화

**Archived:** 2026-05-03
**Status:** All requirements complete

## Milestone v1.0 Requirements

### Preset Template (PRESET)
- [x] **PRESET-01**: 사용자는 연봉 수준 및 투자 스타일(공격적/안정적 등)을 선택하여 프리셋 템플릿을 로드할 수 있다. ✅ Phase 2.1에서 고해상도 데이터로 고도화 완료.
- [x] **PRESET-02**: 템플릿 로드 시 표준 자산 흐름(수입 분배)이 자동 계산되어 즉시 시각화 화면에 반영된다. ✅ Phase 1 완료.
- [x] **PRESET-03**: 사용자는 생성된 템플릿에서 카테고리별 세부 항목을 수동으로 조절하고 변경 사항에 따라 흐름을 재계산할 수 있다. ✅ Phase 2 완료.
