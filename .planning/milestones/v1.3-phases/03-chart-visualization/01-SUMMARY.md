# Plan 01 Summary: 시뮬레이션 차트 인터랙티브 고도화 및 프리셋 UI

## What Was Done
- **차트 반응형 및 Y축 개선:** SVG `viewBox` 속성을 적용해 모바일 환경에서도 유연하게 차트가 축소/확대되도록 반응형 지원 추가. `maxVal`을 `assetNominalTR` 기준으로 변경하고 자산 스케일의 금액(만원 단위)을 표시하는 Y축 그리드와 눈금 텍스트 추가.
- **복리 효과 가시화:** PR(미투자) 선과 TR(재투자) 선 사이를 `polygon` 영역(`fill: rgba(234, 91, 42, 0.1)`)으로 채워, 재투자에 따른 복리 효과 차이가 시각적으로 드러나도록 구현.
- **인터랙티브 툴팁:** 툴팁(`chart-tooltip`)에 Glassmorphism 패널 스타일 적용. 데이터 포인트별로 정확한 호버/터치를 감지하기 위해 차트 내 보이지 않는 투명 수직 `<rect>`(Hover/Touch 영역) 추가. `mouseover`, `touchstart` 발생 시 만원 단위의 연도별 상세 데이터(총 자산, 연 배당금)를 SVG 캔버스 기준 비율을 사용해 플로팅.
- **배당 시뮬레이션 프리셋 3종 추가:** 가정 설정 UI 영역 내에 대표 ETF 조합 프리셋 버튼("단일 100%", "SCHD+QQQI (1:1)", "3종 혼합")을 추가. 각 버튼 클릭 시 기대 배당수익률, 배당 성장률, 주가 상승률 input에 즉시 반영되고 재계산 렌더링을 호출함.
