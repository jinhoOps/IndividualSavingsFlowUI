# Phase 5: Portfolio Creation & Target Allocation UI - Summary

**Executed:** 2026-06-16
**Status:** Complete (v0.11.72)

## 📋 Completed Items

### 1. Portfolio Creator Editor Form (포트폴리오 에디터)
- **동적 종목 편집 폼**: 사용자가 원하는 포트폴리오 이름을 적고, 종목명 및 티커 정보를 담은 행을 동적으로 추가 및 삭제할 수 있는 입출력 폼 구현.
- **가로형 세그먼트 스위치**: 적립 주기를 '매일', '매주', '매달' 단위로 직관적으로 토글 선택할 수 있는 Segment Control 구현.

### 2. 실시간 비중 및 금액 합계 계산 시스템
- **자동 합산 및 비중 산출**: 각 종목의 투자 금액을 입력함에 따라 총 주식 종목 수와 총합 금액이 실시간 연동되어 갱신되며, 총액 대비 각 종목의 비중 %가 실시간 계산되어 정수 단위로 노출.
- **인풋 포커스 유실 방지**: 인풋 포커스가 풀리지 않도록 폼 전체를 리렌더링하지 않고, 수치와 비중 통계 텍스트 영역만 타겟하여 부분 갱신하도록 설계 (`IsfDom.updateCreatorFormStats` 적용).

### 3. 세부 상세 보기 팝업 및 누적 투자액 차트
- **Glassmorphism Detail Modal**: 리스팅된 카드를 클릭하면 팝업 모달이 `backdrop-filter: blur(10px)` 블러 처리된 디자인으로 노출됨.
- **1년 누적 복리 바 차트**: 설정된 주기와 종목 비중에 근거하여 1개월, 3개월, 6개월, 1년차 누적 투입 금액을 복리 시뮬레이션하여 세로 막대 그래프로 가시화.
