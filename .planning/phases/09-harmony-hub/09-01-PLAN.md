# Plan: 09-01 Smart Clipboard Parser Engine

## 📋 Task Overview
은행/카드 결제 문자를 분석하여 금액, 상호명, 날짜를 추출하는 파싱 엔진을 구현합니다. 이 엔진은 사용자의 입력을 자동화하여 Step 1의 지출 항목 업데이트를 용이하게 합니다.

## 🛠 Proposed Changes

### 1. New Module: `shared/core/clipboard-parser.js`
- 다양한 카드사/은행의 문자 포맷을 처리하는 정규표현식 기반의 엔진.
- `parseSms(text)` 함수: 객체 `{ amount, merchant, date, raw }` 반환.
- `matchCategory(merchant, existingItems)` 함수: 가장 유사한 기존 지출 항목 추천.

### 2. UI Integration: Step 1 (`apps/step1/index.html`)
- '생활비 상세 항목' 섹션에 '클립보드 붙여넣기' 버튼 및 입력 팝업 추가.
- 파싱된 결과를 확인하고 기존 항목에 더하거나 새로운 항목으로 추가하는 UI.

### 3. Logic Integration: Step 1 (`apps/step1/app.js`)
- 파서 모듈 임포트.
- 파싱 결과 반영 시 `state.inputs` 업데이트 및 `markPendingChanges` 호출.

## 📅 Execution Steps
1. **Engine Core**: `shared/core/clipboard-parser.js` 작성 및 다양한 테스트 케이스 검증.
2. **UI Scaffolding**: Step 1 HTML에 모달 또는 인라인 입력 필드 추가.
3. **Binding**: 버튼 클릭 시 클립보드 데이터 읽기 (또는 텍스트 입력) 및 파싱 로직 연결.
4. **Validation**: 실제 카드 문자 샘플을 통한 동작 확인.

## 🧪 Testing Strategy
- **Unit Tests**: `clipboard-parser.js`에 대한 다양한 SMS 문자열 테스트.
- **Manual UAT**: 모바일 환경에서 문자 복사 후 ISF 앱에 붙여넣기 동작 확인.
