# Phase 4: Account Transfer UI & UX Polish - Wave 1 Summary

**Executed:** 2026-06-12
**Status:** Verification Ready (v0.11.28)

## 📋 Completed Items

### 1. Account Flow Network Map (계좌 간 자금 흐름 지도)
- **SVG 기반 네트워크 맵 구현**: 계좌 노드(박스)들을 화면 중앙 기준으로 원형 배치(Circular Layout)하여 겹치지 않고 정갈하게 배치.
- **이체선 펄스 애니메이션**: 각 노드를 잇는 SVG path 상에 `@keyframes`와 `stroke-dashoffset` 속성을 이용해 돈이 흐르는 방향성을 부여하는 애니메이션 흐름선 구현.
- **마우스 호버 포커스 필터링**: 특정 계좌 노드에 마우스 진입 시, 해당 노드와 연동되지 않은 모든 링크 및 노드들을 투명도 `0.15`로 낮춰 N:N 관계를 직관화.

### 2. Manual Transfer Rules & UI (수동 이체 설정 및 편집기)
- **데이터 모델 확장**: `Step1State` 모델에 `transfers` 규칙 데이터 스키마를 영속화하여 IndexedDB/localStorage에 보존되도록 확장.
- **문장형 Sentence UI**: 출발 계좌, 도착 계좌, 금액(만원), 라벨을 한 줄로 지정하여 이체 규칙을 추가할 수 있는 간결한 추가 폼 탑재.
- **실시간 출금 가능 예상 잔액 힌트**: 출발 계좌 선택 시, 현재 입력 상태를 기준으로 계산한 남은 잉여 잔고 힌트(#sourceBalanceHint)를 실시간 출력해 뇌적 연산을 최소화.
- **호버 삭제 버튼**: 등록된 이체 규칙 카드는 깔끔한 텍스트로 보이며, 마우스 호버 시에만 X 삭제 버튼이 페이드인으로 나타나도록 정리.

### 3. 시각화 가로 슬라이드 전환 (Visual Slide Transition)
- **단일 영역 토글**: Sankey 차트와 네트워크 맵이 단일 시각화 패널 영역 내에 정렬되어, 탭 토글 클릭 시 `transform` 트랜지션(0.4s cubic-bezier)에 맞춰 가로로 스와이프되어 전환되는 슬라이더 탑재.
- **전역 부유 효과**: Step 1 내의 모든 카드와 리스트 로우 요소에 마우스 호버 시 2px 부유감 및 그림자 트랜지션을 일괄 이식.

### 4. 종합과세 경고 인디케이터 디자인 규격화
- **정량적 스펙 적용**: 금융소득종합과세 한도 초과 경고 인디케이터를 `styles.css`에 규격화하여 분리 구현.
  - *Warn (1,900만 원 초과)*: Yellow 보더, 소프트 box-shadow 글로우 효과.
  - *Crit (3,400만 원 초과)*: Red 보더, box-shadow 글로우 효과.
  - *동작*: 저축/투자 수정 시 레이턴시 없이 cubic-bezier 속성으로 부드럽게 점등/소등.

### 5. 도움말 ? 아이콘 툴팁 통합
- **설명 텍스트 소거**: 화면 내 가독성을 방해하는 긴 안내 텍스트들을 전부 물음표 `?` 아이콘으로 대체.
- **Glassmorphism 툴팁**: 아이콘 호버 시, 마우스 위치에 부드럽게 나타나는 Glassmorphism 팝업 툴팁(#globalTooltip) 구현.

## 👤 Verification Scenarios (UAT Guide)

1. **시각화 패널 토글**:
   - `Sankey 흐름` ↔ `계좌 이체망` 탭을 클릭하여 좌우로 차트가 부드럽게 슬라이드 전환되는지 확인합니다.
2. **수동 이체 설정**:
   - 🏦 계좌 관리 탭 하단에서 수동 이체를 추가/삭제해보고, 출발 계좌 선택 시 아래 힌트 텍스트(`💡 출금 가능 예상 잔액`)가 실시간 변경되는지 확인합니다.
   - 추가된 수동 이체가 `계좌 이체망` 차트에 화살표선 및 펄스 파티클 흐름으로 반영되는지 확인합니다.
3. **설명 툴팁**:
   - 곳곳에 배치된 `?` 아이콘에 마우스를 호버하여 반투명하고 아름다운 도움말 툴팁이 팝업되는지 확인합니다.
4. **금융소득과세 경고**:
   - 저축 탭 이율이나 투자 금액을 1,900만 원 혹은 3,400만 원 금융소득 초과선으로 변경 시, 계좌 카드의 보더 링과 뱃지가 즉각 경고/위험 스펙으로 전환되는지 확인합니다.
