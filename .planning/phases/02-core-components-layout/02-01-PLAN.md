    - **인라인 확장 편집 카드 (D-02)**: 모바일 편집 활성화 시(`.is-editing`) 카드가 세로 3층 구조(1층: 이름, 2층: 계좌+금액, 3층: 적용 및 삭제 버튼)로 미디어 쿼리를 이용하여 깔끔하게 수직 확장 정렬되도록 CSS를 구축합니다.
