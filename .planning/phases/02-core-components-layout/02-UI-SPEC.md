## Visual Hierarchy & Focal Points

### 1. 주요 화면별 포커스 포인트 (Focal Point)
- **모바일 탭 네비게이션 개편 (D-03)**:
  - **1순위 (Primary Focal Point)**: 에디토리얼 언더라인 스타일이 적용된 활성화 탭(`border-bottom: 2px solid var(--income)`) 및 아이콘.
  - **2순위 (Secondary Focal Point)**: 비활성화 상태의 탭 텍스트 및 터치 드래그 감도.
- **모바일 입력 폼 (D-01, D-02)**:
  - **1순위**: 입력창 포커스 시 점등되는 Sunset 오렌지 `#ea5b2a` 또는 Accent 그린 `#1e8b7c` 계열의 **얇은 포커스 링**.
  - **2순위**: 모바일 편집 활성화 시 수직 3단 스택 구조로 아래로 넓게 펼쳐지는 **Inline Expand Card** 영역.
  - **3순위**: 비편집 상태의 플랫 가독 요약 행.

### 2. 요소별 시각적 위계 (Visual Hierarchy)
- **그림자 제거 (Flat Hairline, D-04)**:
  - 모든 카드와 버튼에서 과도한 그림자를 완전히 제거하고 얇고 정밀한 실선 테두리(`1px solid var(--color-line)`)만 남깁니다.
  - 이를 통해 Anthropic 에디토리얼 디자인의 단정함과 지면 레이아웃 특성을 강조합니다.
- **포커스 피드백 효과 (D-04)**:
  - 포커스 시 웜톤 크림배경과 가장 조화로운 브랜딩 Sunset 오렌지 또는 Accent 그린 아웃라인을 부드럽게 점등합니다 (`box-shadow: 0 0 0 2px var(--color-focus-ring)`).
- **입력 필드 반응형 그리드 (D-01)**:
  - 모바일(<= 760px)에서는 1열 세로 스택 구조로 정렬하여 터치 편리성을 제공하고, PC(>= 768px)에서는 3열 그리드로 넓게 분산하여 한눈에 파악할 수 있도록 반응형 위계를 제어합니다.
