# Phase 1 Context: 프리셋 템플릿 로드 및 자동 시각화

## Domain Boundary
사용자가 연봉 수준과 투자 스타일을 선택하고, 이를 기반으로 표준 자산 흐름 템플릿을 자동 생성하여 기존 시각화 엔진에 반영하는 기능.

## Implementation Decisions

### 선택 인터페이스(UI) 구성 방식
- **결정:** 연봉은 드롭다운 방식(예: 3천만 원, 3천 5백만 원 등)으로 선택하고, 투자 성향은 3개의 버튼(안정형, 중립형, 적극형) 중 하나를 클릭하는 방식.
- **노트:** 추후 배당/인덱스/섹터혼합 등 다양한 투자 스타일 템플릿이 추가될 수 있도록 버튼 및 데이터 구조를 확장 가능하게 설계.

### 기존 데이터 덮어쓰기(초기화) 정책
- **결정:** 기존에 사용자가 입력/수정해둔 데이터가 존재할 때 프리셋을 적용하려고 하면 명시적인 확인 팝업(Confirm)을 띄워 덮어쓸지 의사를 묻는다.
- **노트:** 의도치 않은 사용자 데이터 유실 방지.

### 프리셋 데이터 관리/연산 방식
- **결정:** 동적 수식 연산 (Formula-based).
- **노트:** 선택한 연봉에서 기초 수입을 산출하고, 각 투자 성향(안정형/중립형/적극형)에 미리 정의된 비율(비중)을 곱해서 분배 금액을 실시간으로 계산한다.

## Deferred Ideas
- 향후 투자 성향별 다양한 스타일(배당, 인덱스, 섹터혼합 등) 프리셋 추가 예정.
