# Phase 1 Discussion Log

## 1. 선택 인터페이스(UI) 구성 방식
- **Presented:** 드롭다운 + 버튼, 자연어 문장형, 숫자 입력 + 슬라이더
- **Selected:** 드롭다운(연봉) + 버튼 3개(투자 성향)
- **Notes:** 추후 다양한 투자 스타일을 추가할 수 있도록 확장성 고려.

## 2. 기존 데이터 덮어쓰기(초기화) 정책
- **Presented:** 완전 덮어쓰기(스낵바), 명시적 확인 창(Confirm), 소프트 리셋(병합)
- **Selected:** 명시적 확인 창
- **Notes:** 기존 입력 데이터가 있을 때만 경고 팝업 표시.

## 3. 프리셋 데이터 관리/연산 방식
- **Presented:** 동적 수식 연산, 정적 데이터 맵핑
- **Selected:** 동적 수식 연산
- **Notes:** 투자 성향에 정의된 비율을 사용하여 계산.
