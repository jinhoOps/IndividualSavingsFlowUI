# Plan: 10-01 AI Integration Architecture & Core Engine

## 📋 Task Overview
AI 자문 기능을 위한 클라이언트 사이드 아키텍처를 설계하고, Google Gemini API를 연동하여 기본적인 인사이트 추출 엔진을 구현합니다.

## 🛠 Proposed Changes

### 1. New Module: `shared/core/ai-engine.js`
- Google Gemini API (또는 OpenAI) 연동 추상화 레이어.
- `generateInsight(data)`: 시뮬레이션 데이터를 요약하여 텍스트 인사이트 생성.
- `answerTaxQuestion(query, currentPortfolio)`: 세무 정책 기반 대화 기능.

### 2. UI: AI Integration Setup
- `DataHubModal` 내 'AI 설정' 섹션 추가 (API 키 입력 필드).
- API 키의 로컬 스토리지 보안 저장 (암호화는 범위 밖, 로컬 전용).

### 3. Logic: Data Preparation
- `state.inputs`와 시뮬레이션 결과를 AI가 읽기 좋은 JSON 스키마로 변환하는 유틸리티 구현.

## 📅 Execution Steps
1. **Engine Core**: `ai-engine.js` 작성 및 API 호출 연동.
2. **Setup UI**: API 키 관리 인터페이스 추가.
3. **Prompt Engineering**: 가계 분석 및 세무 자문을 위한 시스템 프롬프트 정의.
4. **Validation**: 테스트 데이터를 이용한 인사이트 생성 품질 확인.

## 🧪 Testing Strategy
- **Manual UAT**: 서로 다른 자산 상태(예: 고배당 포트폴리오 vs 고지출)에서 AI가 적절한 인사이트를 생성하는지 확인.
- **Safety Check**: 잘못된 API 키 입력 시의 에러 핸들링 검증.
