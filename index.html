Individual Savings Flow

개인 자산흐름, 한 장으로

핵심 카드는 유지하고, 하단 그래프는 단일 생키차트로 통합했습니다. 자산 추이는 같은 계산 엔진으로 월 단위 시뮬레이션해 검증 테이블로 제공합니다.

핵심 카드

월 자산흐름 생키차트

표시할 흐름이 없습니다.

자산 추이 계산 검증

순자산 = 현금 + 저축 + 투자 - 부채
매월: (수입 - 생활비 - 저축 - 투자 - 실제 부채상환액)을 현금에 반영합니다. (실제 부채상환액 = 설정 상환액, 부채잔액, 가용현금 중 최솟값) 저축/투자는 가용현금 범위 내에서만 반영됩니다. 부채는 월 이자를 먼저 반영하고 상환액으로 차감하며, 현금이 음수면 부족분을 부채에 더합니다. 저축/투자에는 월 수익률을 적용합니다. 실질 순자산(현재가치)은 기준금리(연)를 월 복리로 환산해 할인합니다.
시점 월 수입 월 생활비 당월 이자 실제 상환액 부채 증가분 현금 저축 투자 부채 순자산 실질 순자산(현재가치)

입력값

금액 단위: 만원(예: 450 = 4,500,000원) · 기본값만 입력해도 바로 계산됩니다.

월 수입 항목

수입 항목은 여러 개 추가할 수 있고, 자동 합산됩니다.

간편 입력

고급 설정 (생활비 상세 / 초기자산 / 증가율 / 수익률)

생활비 상세 항목

아래 항목 합계가 월 생활비로 자동 반영됩니다.

저축 상세 항목

아래 항목 합계가 월 저축으로 자동 반영됩니다.

투자 상세 항목

아래 항목 합계가 월 투자로 자동 반영됩니다.
