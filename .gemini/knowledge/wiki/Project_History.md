# Project History (프로젝트 연대기)

## 🕒 주요 마일스톤

### [2026-06-16] Milestone v1.8: 적립식 포트폴리오 관리 고도화 (v0.11.83) 📈
- **주요 내용**: Step 3 적립식 포트폴리오 관리 핵심 기능 및 사용자 상태 제어 고도화.
- **성과**:
    - 포트폴리오 만들기/편집기 UI 완성 및 매일/매주/매월 프리미엄 세그먼트 주기 선택기 도입.
    - 1,000원 단위 +/- 가감 시 수학적 올림/내림 자동 보정 공식 적용.
    - 데이터 변경 감지 시 3종 랜덤 애니메이션(`anim-slide-up` 등) 플로팅 펜딩 바(Pending Bar) 노출 및 취소 시 이어서 편집 UX 정립.
    - 상시 활성화 생성 버튼 및 미입력/오입력 원인을 콕 집어 피드백하는 디테일 경고창 연동.
    - 매일 적립 시 실제 금융 영업일(연 240일) 기준으로 투입액을 환산하여 현실적인 1년 누적 투자 추이를 차트로 표출.
    - 포트폴리오 요약 카드의 호버 테두리 Sunset Orange 광채 및 Y축 -2px 부유 효과 적용.

### [2026-06-15] 1원 단위 입력 전면 통일 및 실시간 금액 힌트 (v0.11.46) 🪙
- **주요 내용**: 만원/원 단위 혼재로 인한 복리 연산 오류를 해결하고자 전 시스템을 1원 단위로 전면 단일화.
- **성과**:
    - `toWon` 및 `toMan` 함수를 1:1 통과 방식으로 패치하여 만원 변환 버그 원천 봉쇄.
    - 사용자의 인풋 입력 편의성을 유지하기 위해 입력 숫자 감지 즉시 한글 금액으로 실시간 변환해주는 힌트(`.realtime-won-hint`) 상시 표출.

### [2026-06-12] Milestone v1.7: 다중 계좌 매핑 & 이체 시각화 (v0.11.35) 🔗
- **주요 내용**: 통장 쪼개기 흐름을 설계하는 다중 계좌 데이터 모델 반영 및 SVG N:N 계좌이체망 도입.
- **성과**:
    - 수입/지출 항목을 커스텀 계좌(급여, 생활비, 주식, CMA)에 직접 연결하는 4단계 생키(Sankey) 레이어 고도화.
    - 남는 잉여현금을 자동으로 지정한 주식 계좌 등으로 분배하는 문장형 자동 이체 배너 제공.
    - 계좌 간의 다대다 이체 흐름을 SVG 경로 스트로크 애니메이션을 동반한 네트워크 맵(Network Map)으로 시각화.
    - 생키 차트와 네트워크 맵 간의 가로 슬라이딩 스위칭 전환 탭 연동.
    - 종합과세 경고(1,900만/3,400만)를 화면 상단 헤더의 단일 LED 인디케이터로 깔끔하게 통합하여 인지 부하 감소.
    - 도움말 ? 아이콘을 도입해 전역 Glassmorphism 툴팁 내부로 긴 도움말 텍스트 축소 통합.

### [2026-06-10] Milestone v1.6: 코드 리팩터링 및 UI 안정성 강화 (v0.11.2) 🛠️
- **주요 내용**: 모듈화 리팩터링 과정에서 발생한 잔여 오류 제거 및 모바일 UI 접근성 강화.
- **성과**:
    - 런타임 ReferenceError 우려가 있었던 4종의 레거시 유령 함수 전수 복구.
    - 온보딩 가이드 툴팁에 우측 상단 X 닫기 버튼을 주입해 즉시 스킵을 허용하고, 헤더 내 튜토리얼 다시보기 기능 마련.
    - 모바일 760px 해상도에서의 아코디언 접기 및 수직 1열 레이아웃을 엄격히 조율해 반응형 무결성 보장.

- **인프라 혁신**: Vite, TypeScript, TailwindCSS v4 도입 및 레거시 호환성 브릿지 구축 완료.

- **시스템 무결성**: Branded Types 도입 및 단위 정합성(만원/원) 수호 로직 강화.
