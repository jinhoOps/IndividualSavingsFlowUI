# Financial Taxation Reference (금융소득 및 과세 정책 참조)

본 문서는 IndividualSavingsFlowUI 프로젝트에서 적용하는 금융소득(이자/배당)에 대한 과세 논리와 UI 경고 정책을 정의합니다.

## ⚖️ 대한민국 금융소득종합과세 정책 (요약)

은행 예금 이자나 주식 배당금 등 금융소득을 합산하여 **연 2,000만 원**이 초과될 경우 금융소득종합과세 대상이 됩니다.

- **분리과세 (2,000만 원 이하):** 15.4% (지방소득세 포함) 세율의 원천징수로 종결.
- **종합과세 (2,000만 원 초과):** 금융소득을 사업/근로소득 등 다른 종합소득과 합산하여 누진세율(6% ~ 45%) 적용.
- **최저한세 원칙:** 종합과세 시의 세액이 분리과세(14% + 지방소득세)보다 적지 않도록 `MAX[일반산출세액, 비교산출세액]` 방식을 적용합니다.

### 종합소득 누진세율 (2023년 귀속분 기준)

| 과세표준 구간 | 세율 | 누진공제 |
|---|---|---|
| 1,400만 원 이하 | 6% | - |
| 1,400만 원 초과 5,000만 원 이하 | 15% | 126만 원 |
| 5,000만 원 초과 8,800만 원 이하 | 24% | 576만 원 |
| 8,800만 원 초과 1.5억 원 이하 | 35% | 1,544만 원 |
| 1.5억 원 초과 3억 원 이하 | 38% | 1,994만 원 |
| 3억 원 초과 5억 원 이하 | 40% | 2,594만 원 |
| 5억 원 초과 10억 원 이하 | 42% | 3,594만 원 |
| 10억 원 초과 | 45% | 6,594만 원 |

*참고: 지방소득세 10% 별도 부과 (실질 세율 6.6% ~ 49.5%)*

## 🚨 프로젝트 UI 경고 정책 (Safety Margin)

실제 법적 기준보다 보수적으로 접근하여 사용자가 미리 대비할 수 있도록 **4%의 안전 마진(Safety Margin)**을 설정합니다.

### 1. 금융소득종합과세 주의 (`warn`)
- **기준:** 연간 이자/배당 소득 합계 **1,920만 원** 초과 시.
- **논리:** 법정 기준(2,000만 원)에 대해 **4%의 마진**($2,000 \times 0.96$)을 두어 보수적으로 경고를 띄웁니다.
- **UI 피드백:** 노란색 경고 아이콘 및 "과세주의" 메시지 표시.

### 2. 치명적 경고 (`crit`)
- **기준:** 연간 이자/배당 소득 합계 **3,264만 원** 초과 시.
- **논리:** 과거 고액 소득 기준(3,400만 원)에 대해 **4%의 마진**($3,400 \times 0.96$)을 적용하여 강력한 주의가 필요한 구간으로 설정합니다.
- **UI 피드백:** 빨간색 경고 아이콘 및 "과세경고" 메시지 표시.

## 🛠️ 구현 가이드

- `IsfUtils` 또는 전용 `TaxCalculator` 모듈을 통해 위 기준값을 상수로 관리하십시오.
- 모든 계산은 내부적으로 `원` 단위를 사용하며, UI 표시 시에만 `만원` 또는 `억/만원` 단위로 변환합니다.
