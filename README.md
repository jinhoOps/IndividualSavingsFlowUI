# IndividualSavings Flow (ISF) - 개인 자산 흐름 시각화 도구

> **"복잡한 가계 관리와 자산 흐름을 한눈에 보고 직관적으로 의사결정하기 위한 미니멀 UI/UX 도구"**

`IndividualSavings Flow (ISF)`는 사용자가 복잡한 재무 지식이나 설정 없이도, 단순한 입력(연봉, 투자 성향 프리셋 선택)만으로 자신의 전체 가계 현금 흐름과 저축/투자 비중을 직관적인 **Sankey Diagram**으로 시각화할 수 있도록 돕는 정적 웹 기반 자산 관리 도구입니다.

데이터 수집이나 서버 영속화를 최소화하여 개인정보 노출 걱정 없이 오프라인에서도 강력하게 동작하며, 철저한 금융 연산 규칙과 안정적인 성능을 자랑합니다.

---

## 🎯 프로젝트 의도 & 철학

* **직관적인 예산 흐름**: 수입이 어떤 계좌를 거쳐 지출, 저축, 투자로 쪼개지는지 가시적인 4단계 레이어(수입원 ➔ 계좌 ➔ 세부항목) 흐름으로 한눈에 파악합니다.
* **오프라인 퍼스트 & 데이터 무결성**: 서버 통신을 배제한 브라우저 내부 스토리지(IndexedDB)와 백업 기능을 통해 철저히 개인화된 데이터 흐름을 유지합니다.
* **사용자 경험(UX) 중심**: 유려한 Glassmorphism 및 Sunset/Deep Sea 브랜드 포인트를 담은 디자인 시스템, 모바일 기기를 고려한 반응형 UI를 통해 쓰기 편하고 시각적으로 만족스러운 환경을 제공합니다.
* **금융 정합성 수호**: UI는 사용자가 체감하기 쉬운 '만원' 단위로 일관되게 표시하며, 데이터 처리 및 저장 시에는 '원' 단위를 유지해 1원의 오차도 허용하지 않는 엄격한 금융 연산 규칙을 구현합니다.

## ✨ 구현된 기능 (Shipped Features)

### 1단계: 나의 가계 흐름 (Step 1)
* **Spotlight 온보딩 가이드**: 신규 사용자도 서비스의 기능을 빠르게 탐색할 수 있는 시각적 집중 온보딩 시스템.
* **수입/지출 통장 쪼개기 (계좌 관리)**: 단순 금액 입력을 넘어 급여계좌, 생활비계좌, 주식계좌 등으로 세부 항목들을 매핑하는 4단계 Sankey 연산 엔진 탑재.
* **잉여현금 자동 이체 배너**: 수입에서 지출/저축/투자를 뺀 잔여 금액을 지정한 투자 계좌로 자동으로 이체하는 문장형 UI 및 실시간 차트 업데이트.
* **금융소득종합과세 경고**: 연간 이자/배당 소득이 기준치를 초과할 경우(1,900만 원 초과 시 주의, 3,400만 원 초과 시 경고)를 감지하는 안전 마진 알림 제공.

### 2단계: 배당 성장 시뮬레이션 (Step 2)
* **배당 재투자(DRIP) 연산 엔진**: 자산 성장률(CGR), 배당 성장률(DGR), 배당 재투자를 결합한 30개년 자산 성장 시뮬레이터.
* **인플레이션 보정 계산**: 실질 가치를 연산하여 미래의 자산 가치를 현실적인 금액으로 변환해 차트 및 표로 시각화.
* **Step 1 자동 데이터 연동**: 1단계에서 산출된 월 투자 여력과 초기 자산 데이터를 즉각적으로 시뮬레이션에 반영.

### 데이터 및 플랫폼 인프라
* **고해상도 이미지 내보내기**: 화면에 렌더링된 Sankey 차트를 2배 고해상도 PNG 파일로 깨짐 없이 다운로드 및 공유할 수 있는 캔버스 이미지 내보내기.
* **IndexedDB 기반 스토리지 v2**: 안전한 로컬 저장과 12시간 자동 백업 시스템.
* **유연한 데이터 백업/복원**: 데이터 허브를 통한 구버전 데이터 호환 마이그레이션(하이드레이션), 부부 데이터 병합 및 URL 해시(#s) 공유 링크.
* **PWA(Progressive Web App)**: 모바일 및 데스크톱 홈 화면 추가를 지원하며, 오프라인 환경에서도 서비스 완전 사용 가능.

## 🛠️ 설치 및 시작하기

본 프로젝트는 Vite 빌드 시스템을 사용합니다.

```bash
# 1. 저장소 복제 및 이동
git clone https://github.com/jinhoOps/IndividualSavingsFlowUI.git
cd IndividualSavingsFlowUI

# 2. 패키지 설치
npm install

# 3. 개발 서버 실행
npm run dev

# 4. 프로덕션 빌드 (빌드 완료 시 패치 버전이 자동 범프됩니다)
npm run build
```
