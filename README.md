# IndividualSavingsFlowUI

개인 자산 흐름을 한 화면에서 보는 웹 도구입니다.  
설치 없이 `index.html`만 열면 바로 사용할 수 있습니다.

## 1) 문제 정의

- 월 수입/지출/저축/투자/부채 정보를 여러 앱에 나눠 관리하면 전체 흐름을 한눈에 보기 어렵습니다.
- 특히 "이번 달 돈이 어디로 흘렀는지"와 "앞으로 자산이 어떻게 변하는지"를 빠르게 확인하기 어렵습니다.

## 2) 목표

- 입력은 최대한 단순하게 유지하고, 결과는 카드 + Sankey + 검증표로 직관적으로 보여줍니다.
- 부채 이자/상환/증가분까지 함께 노출해 계산 근거를 이해하기 쉽게 만듭니다.
- 일반 사용자도 설치 없이 바로 실행해 개인 자산 흐름을 점검할 수 있게 합니다.

## 3) 바로 실행

```bash
git clone https://github.com/jinhoOps/IndividualSavingsFlowUI.git
cd IndividualSavingsFlowUI
```

파일 탐색기에서 `index.html` 더블클릭하면 바로 실행됩니다.

## 4) 무엇을 할 수 있나요?

- 월 수입/생활비/저축/투자/부채상환 입력
- 핵심 카드로 현재 상태 요약 확인
- Sankey 차트로 돈 흐름 한눈에 확인
- 자산 추이 계산 검증 표 확인
- 실질 순자산(현재가치)까지 확인
- 고급설정에서 생활비/저축/투자 상세 항목 편집

## 5) 사용 흐름 (간단)

1. 하단 `입력값`에서 금액 입력
2. 필요하면 `고급 설정` 열기
3. 상세 항목 구조를 바꾸려면 `항목 편집` 클릭 후 적용
4. 입력 변경 후 하단 `적용/취소`로 반영
5. 상단 카드 + Sankey + 검증표로 결과 확인

## 6) 기본값

- 월 수입: 300 (만원)
- 생활비/저축/투자 기본 상세 항목 제공
- 초기 자산/부채 및 수익률 기본값 제공

## 7) 주의

- 금액 단위는 `만원`입니다. (예: 300 = 3,000,000원)
- 브라우저 로컬 저장소를 사용해 입력값이 유지됩니다.
- 완전 초기화는 `초기화` 버튼을 사용하세요.
