# 나의 가계 흐름

이렇게 벌고, 이렇게 쓴다.  
설치 없이 `index.html`만 열면 바로 사용할 수 있습니다.

`개인 자산 흐름 (IndividualSavings Flow UIUX)` 프로젝트의 1단계(스켈레톤) 화면입니다.

## 1) 문제 정의

- 월 수입/지출 정보를 여러 앱에 나눠 관리하면 이번 달 돈의 흐름을 한눈에 보기 어렵습니다.
- 특히 "얼마를 벌고, 어디에 썼는지"를 빠르게 점검하기 어려워 가계 판단이 늦어집니다.

## 2) 목표

- 월 단위 `수입 → 생활비/저축/투자/부채상환 → 잔여현금` 흐름을 직관적으로 보여줍니다.
- 입력은 단순하게 유지하고, 카드 + Sankey + 검증표로 계산 근거를 명확하게 제공합니다.
- 개인 자산 흐름 프로젝트 확장을 위한 UIUX 기반(첫 단계)을 안정적으로 만듭니다.

## 3) 바로 실행

```bash
git clone https://github.com/jinhoOps/IndividualSavingsFlowUI.git
cd IndividualSavingsFlowUI
```

파일 탐색기에서 `index.html` 더블클릭하면 바로 실행됩니다.

또는 배포된 페이지 `https://jinhoops.github.io/IndividualSavingsFlowUI/`로 바로 접속해 사용해도 됩니다.

## 4) 무엇을 할 수 있나요?

- 월 수입/생활비/저축/투자/부채상환 입력
- 핵심 카드로 현재 상태 요약 확인
- Sankey 차트로 월 가계 흐름 한눈에 확인
- 가계 추이 계산 검증 표 확인
- 실질 순자산(현재가치)까지 확인
- 고급설정에서 생활비/저축/투자 상세 항목 편집
- 링크 복사로 현재 상태를 URL 해시로 공유
- JSON 저장/불러오기로 2차 백업 관리

## 5) 사용 흐름 (간단)

1. 하단 `입력값`에서 금액 입력
2. 필요하면 `고급 설정` 열기
3. 상세 항목 구조를 바꾸려면 `항목 편집` 클릭 후 적용
4. 입력 변경 후 하단 `적용/취소`로 반영
5. 상단 카드 + Sankey + 검증표로 결과 확인

## 6) 기본값

- 월 수입: 300 (만원)
- 생활비/저축/투자 기본 상세 항목 제공
- 초기 자산/부채 및 수익률 기본값 제공

## 7) 주의

- 금액 단위는 `만원`입니다. (예: 300 = 3,000,000원)
- 브라우저 로컬 저장소를 사용해 입력값이 유지됩니다.
- 완전 초기화는 `초기화` 버튼을 사용하세요.
